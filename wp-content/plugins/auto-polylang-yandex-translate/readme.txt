Автоматическое создание и обновление переводов через Yandex Translate API для Polylang Pro.

== Description ==

Плагин для автоматического перевода контента WordPress с использованием Yandex Translate API и интеграцией с Polylang Pro.

== Основные возможности ==

* Автоматический перевод записей, страниц и таксономий
* Обновление существующих переводов при изменении оригинала
* Поддержка пользовательских типов записей и таксономий
* Перевод ACF полей (текст, галереи, изображения, файлы)
* Копирование и связывание изображений через Polylang
* Массовый перевод через Bulk Actions и интерфейс настроек
* Поддержка repeater и flexible content полей ACF

== Installation ==

1. Загрузите папку плагина в `/wp-content/plugins/`
2. Активируйте плагин через меню 'Плагины' в WordPress
3. Убедитесь, что активирован Polylang Pro
4. Получите API ключ Yandex Cloud
5. Настройте плагин в Настройки → Yandex Auto Translate

== Frequently Asked Questions ==

= Как получить Yandex Translate API ключ? =

1. Зарегистрируйтесь в Yandex Cloud
2. Создайте платежный аккаунт
3. Активируйте сервис "Translate API"
4. Получите OAuth-токен в разделе IAM

= Поддерживаются ли пользовательские таксономии? =

Да, плагин автоматически поддерживает все публичные пользовательские таксономии.

= Как обновить существующие переводы? =

Включите опцию "Обновление переводов" в настройках плагина.
