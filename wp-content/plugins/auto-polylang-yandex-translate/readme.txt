Автоматическое создание и обновление переводов через Yandex Translate API для Polylang Pro.

== Description ==

Плагин для автоматического перевода контента WordPress с использованием Yandex Translate API и интеграцией с Polylang Pro.

== Основные возможности ==

* Автоматический перевод записей, страниц и таксономий
* Обновление существующих переводов при изменении оригинала
* Поддержка пользовательских типов записей и таксономий
* Полная поддержка ACF полей (текст, галереи, изображения, файлы, repeater, flexible content, post object, taxonomy)
* Копирование и связывание изображений через Polylang
* Массовый перевод через Bulk Actions и интерфейс настроек
* Ручной перевод через метабоксы
* Пакетная обработка тысяч записей
